## 📦 Installation

```bash
1. Clone repository
```bash
git clone https://github.com/CusJhon/whatsapp-rvo-bot-jhon.git
cd whatsapp-rvo-bot-jhon

```

2. *npm install*

3. Setup environment
Buat file .env lalu isi:

```env
GITHUB_TOKEN=your_github_token
REPO=username/repository
BRANCH=main
PORT=3000
```

4. Jalankan aplikasi

```Bash

npm start

```

5. Buka di browser

```Bash

http://localhost:3000

```

6. ```Cara pakai```
# Upload file melalui web
# File otomatis di-upload ke repository # GitHub
# Commit dilakukan secara otomatis

## 🚀 Usage

```bash
---

> Note: Ini bukan bot WhatsApp asli, command hanya representasi fitur.

### 📂 Upload File
.upload <file>
Upload file ke repository GitHub

### 📁 Upload ke Folder
.upload <file> /folder/
Upload file ke path tertentu dalam repo

### 🌿 Set Branch
.branch <name>
Mengganti target branch

### 📜 Set Repository
.repo <username/repo>
Menentukan repository tujuan

### 🔑 Set Token
.token <github_token>
Mengatur akses GitHub

### 📝 Commit Message
.commit <message>
Mengatur pesan commit

### 📊 Status
.status
Menampilkan konfigurasi saat ini
```
